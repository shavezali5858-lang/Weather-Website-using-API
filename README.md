# 🌦️ Weather Website (API-Based)

## 📌 Overview

This project is a simple weather website that allows users to check real-time weather information by entering a city name. It fetches live data from a weather API and displays key details like temperature, humidity, and weather conditions.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature
* 💧 Shows humidity levels
* ☁️ Weather condition (Clear, Clouds, Rain, etc.)
* ⚠️ Error handling for invalid city names
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Weather API

---

## ⚙️ How It Works

1. User enters a city name
2. JavaScript sends a request to the weather API
3. API returns real-time weather data
4. Data is displayed dynamically on the webpage

---



---

## 🔑 API Used



* OpenWeatherMap API

## 🧑‍💻 How to Run Locally

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
